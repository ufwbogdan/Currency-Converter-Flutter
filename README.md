# A Currency Converter app using Flutter

  - A currency converter app that uses a tracking locator and an API to fetch real time exchange rates.
  - The app uses Geolocator on startup so that the currency set in the home page matches the one of the user's location.
  - The app requires an API key from https://www.exchangerate-api.com/, and should be run using flutter run --dart-define=EXCHANGE_RATES_API=your_api_key_here.
  - Using Flutter, the app works on multiple types of devices such as IOS, Android and Web.
